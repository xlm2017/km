# nrm


nrm 是一个 npm 源管理器，允许你快速地在 npm源间切换。


# nrm install nrm -g


nrm ls  nrm list


nrm current   查看当前源

nrm use taobao  切换源

nrm add cpm http://sdff.sff...

nrm del cpm 


nrm test npm  测试源速度
