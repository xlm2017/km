# 5选3根据组合公式：
C(5, 3) = A(5, 3) / 3 = ((5 * 4 * 3 * 2 * 1) / (2 * 1)) / (3 * 2 * 1)=10种。

# 5选2根据组合公式：
C(5, 2) = A(5, 2) / 2 = ((5 * 4 * 3 * 2 * 1) / (3 * 2 * 1)) / (2 * 1)=10种。
